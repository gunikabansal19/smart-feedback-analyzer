🤖 What is Smart Feedback System?

Smart Feedback System (SFS) is a web-based feedback collection and analysis platform built using PHP and MySQL. It empowers organizations, institutions, or service providers to gather, analyze, and visualize user opinions with real-time sentiment detection and categorized insights.

🌟 Key Features:

  • User authentication and role-based dashboard
  • Feedback submission with rating and category
  • Sentiment analysis (positive, negative, neutral)
  • Admin analytics with Chart.js visualizations
  • Company-specific views (if enabled)
  • CSV export and filterable feedback history
