Smart Feedback System (SFS) is a powerful, AI-integrated platform for collecting and analyzing user feedback in real-time. Built using PHP, MySQL, and web technologies, it enables businesses and institutions to extract insights and improve services effectively.

Key Features:

  • User roles: Admin, Company, and General Users with personalized dashboards
  • Feedback submission: Rate, comment, and categorize feedback
  • Real-time sentiment analysis: Detects Positive, Negative, or Neutral sentiment
  • Insightful visualizations: Charts and emotion insights powered by Chart.js
  • Data export: Download feedback as CSV with filters
  • Company-specific control: Assign users and manage company data
